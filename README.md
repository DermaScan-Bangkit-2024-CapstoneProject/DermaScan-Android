# **DermaScan Flutter ver Project**

## **Description**

A simple Flutter project. Currently, it has a basic homepage, and I've started organizing assets and fonts for future development.

## **What I've done so far**

- Created a homepage screen.
- Added an empty folder structure for **assets** and **fonts**.
- Created a back button (though the app is still very basic at the moment).

## **How to run the app**

1. Make sure you have Flutter installed on your machine.
2. Clone the repo.
3. Run `flutter pub get` to get the dependencies.
4. Run the app with `flutter run`.

## ** Edited Folder Structure**

- `lib/`: Contains the main app code.
  - `home.dart`: The homepage of the app.
- `assets/`: Folder for images and other static resources (empty for now).
- `fonts/`: Folder for custom fonts (empty for now).
